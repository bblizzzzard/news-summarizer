# Задание 2 — саммаризация новостей через LLM API
## Описание задания
Пайплайн **данные → LLM → структурированный результат**. 
Скрипт читает новости из CSV, отправляет текст в языковую модель через API и получает краткое содержание в формате JSON. Результаты хранятся в `output/results.json`.

## Как работает скрипт
Скрипт читает новости из `data/news.csv` (колонки `id`, `title`, `text`). Для каждой новости формирует запрос и отправляет его в LLM. Модель возвращает JSON: краткое содержание (`summary`), ключевые тезисы (`key_points`) и рубрику (`category`). В конце собирает все ответы и сохраняет в `output/results.json`.

## Инструкция запуска
Для запуска необходимо установить зависимости:
   ```
   pip install -r requirements.txt
   ```
Создать файл `.env` (пример: `.env.example`) и вписать свой ключ (получить ключ на https://openrouter.ai/keys):

   ```
   OPENROUTER_API_KEY = ваш_ключ_сюда
   MODEL = cohere/north-mini-code:free
   ```
Запуск скрипта:

   ```
   python script_summarizer.py
   ```

## Пример входных данных (`data/news.csv`)
